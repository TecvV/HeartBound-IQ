# HeartBound IQ — Multi Agentic AI Wedding Planner

## Features

- Multi-agent wedding planning workflow that runs Venue, Catering, Vendor, Budget, Timeline, Guest, and Orchestrator agents from a single brief.
- Orchestrator that builds a complete plan, auto-picks top vendors, and keeps selections, confirmations, and budgets in sync.
- Venue Scout Agent that surfaces top venues, scores them, and presents AI-ranked picks with pricing, capacity, ratings, and amenities.
- Catering Agent that compares menus, cuisines, packages, and pricing to match guest count and budget.
- Vendor Agent for photographers, decorators, DJs/music, and other services with AI scoring and category-wise picks.
- Decision Review dashboard that compares AI recommendations vs user selections and highlights differences.
- Compare Mode for side-by-side comparisons of vendors and venues.
- Budget Tracker with expected totals, paid/unpaid tracking, and category-wise expense management.
- Timeline Agent that auto-generates a complete day-of schedule and key milestones.
- Guest Management system with guest list, RSVP status tracking, dietary preferences, and invitation scope (guest only vs guest + family).
- RSVP system with a public RSVP link per guest that updates the main guest list automatically.
- Invitation Card designer with multiple themes, live preview, and PDF generation.
- Exportable Wedding Plan PDF and downloadable wedding website PDF for sharing and records.
- Wedding Website generator with auto-filled details from confirmed vendors and finalized selections.
- Invite-all workflow that sends the finalized invitation card PDF and wedding details PDF to every guest.
- Tools gating that unlocks advanced steps only after required confirmations are complete.

## Advanced AI Enhancements

- WhatsApp invitation delivery using WhatsApp Business API, enabling high-open-rate invites and RSVP replies inside WhatsApp that update the guest list instantly.
- Vendor review sentiment analyzer that reads top reviews and surfaces concrete insights (e.g., repeated service issues or praised strengths) beyond star ratings.
- Weather-aware Timeline Agent that checks the wedding date forecast and auto-adds contingency steps for outdoor events.
- Multi-city comparison agent for destination weddings, running parallel plans across cities with cost and quality comparisons.
- Post-wedding photo delivery that matches guest faces and auto-sends their photos through email and WhatsApp.
- AI Voice Agent that contacts vendors, logs conversations, extracts key details, and feeds those insights into final vendor scoring and recommendations.
